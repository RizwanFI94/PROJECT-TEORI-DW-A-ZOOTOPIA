# ZOOTOPIA – Habitat Animals Website

Project ini adalah website edukasi tentang berbagai habitat hewan di dunia, terinspirasi dari dunia Zootopia.  
Pengunjung bisa mengeksplorasi hewan berdasarkan jenis habitat: **Tropical, Arid, Polar, dan Aquatic.**

---

## Struktur Project

```text
.
├─ index.html              # Halaman utama Zootopia
├─ Tropical.html           # Halaman habitat Tropical
├─ Arid.html               # Halaman habitat Arid / gurun
├─ Polar.html              # Halaman habitat Polar / kutub
├─ Aquatic.html            # Halaman habitat Aquatic / perairan
│
├─ style-page/
│   ├─ style-tropical.css  # Style khusus halaman Tropical
│   ├─ style-arid.css      # Style khusus halaman Arid
│   ├─ style-polar.css     # Style khusus halaman Polar
│   └─ style-aquatic.css   # Style khusus halaman Aquatic
│
├─ script-page/
│   ├─ script-tropical.js  # Data & interaksi hewan Tropical
│   ├─ script-arid.js      # Data & interaksi hewan Arid
│   ├─ script-polar.js     # Data & interaksi hewan Polar
│   └─ script-aquatic.js   # Data & interaksi hewan Aquatic
│
├─ Tropical-img/           # Gambar hewan & hero Tropical
├─ Arid-img/               # Gambar hewan & hero Arid
├─ Polar-img/              # Gambar hewan & hero Polar
└─ Aquatic-img/            # Gambar hewan & hero Aquatic
